出生点规划

玩家可以申请一个坐标作为出生点，需要满足以下条件：

  • 位置不能是海洋、地底、天空等无法正常生存的地形
  • 出生点附近不能有过多领地的限制
  • 禁止在出生点设置领地困住玩家

申请流程： 想要申请出生点的请@群主@管理员进行审核，审核过关将会在2天内把出生点转移至建筑

注意事项： 审批通过后出生点会有两周固定时间，两周内玩家的申请将不会进行受理

违规处罚： 如果发现有在出生点设置领地困住玩家的行为，将会进行封禁2天处理

服务器指令

  • /tps - 查询服务器延迟
  • /pland - 创建领地
  • /ping - 查询个人延迟

领地使用方法

领地系统【PLand】提供以下指令：

  • /pland new [default/sub_land] - 创建领地
  • /pland cancel - 取消领地创建
  • /pland set <a/b> - 设置领地位置
  • /pland buy - 购买领地
  • /pland gui - 打开领地UI面板

注意： 由于服务器是原版生存，所以没有开启领地传送功能，无法使用 /pland set teleport_pos 指令
